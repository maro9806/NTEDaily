NTE(Neverness to Everness) 이상관리국 에이전트들을 위한 단일 HTML 기반 데일리 트래커임.
복잡한 설치 필요 없고, 브라우저에 즐겨찾기 해두거나 로컬에 파일 하나 던져두고 쓰면 된다.

## ✨ 주요 기능

* **3종 리셋 시스템:** 일간(매일), 주간(매주 월요일), 고정 숙제.
* **로컬 저장:** DB 그딴 거 필요 없음. 브라우저 `localStorage`에 저장되니까 창 닫아도 안 날아감.

## 🚀 사용 방법
1. `NTE_Daily.html` 파일을 다운로드한다.
2. 브라우저(크롬, 엣지 등)로 파일을 연다.
3. 끝
